
# 团协同是什么？

  

团协同即指团队协同编辑，支持团队成员同时编辑图片，同时支持私人图库空间。


## 如何拉取项目

  

复制Https链接或者下载zip本地源代码压缩包

```

https://github.com/77lwx/Team-Gallery.git

```
